# Satellite Monitoring of TSF Seepage | El Soldado, Chile 🇨🇱

## 🇬🇧 Project Overview (In Development)
This project develops a remote sensing workflow to detect sub-visual vegetation stress as a proxy for seepage at the El Soldado Tailings Storage Facility. 

**Technical Focus:**
* **Platform:** Google Earth Engine (GEE)
* **Data:** Sentinel-2 Multispectral Imagery
* **Methodology:** Red Edge Position (REP) indices and time-series analysis.
* **Compliance:** Aligned with the Global Industry Standard on Tailings Management (GISTM).

---

## 🇨🇱 Resumen del Proyecto (En Desarrollo)
Desarrollo de un flujo de trabajo de teledetección para detectar estrés vegetativo sub-visual como indicador de filtraciones en el Depósito de Relaves El Soldado.

**Enfoque Técnico:**
* **Plataforma:** Google Earth Engine (GEE)
* **Datos:** Imágenes multiespectrales de Sentinel-2
* **Metodología:** Índices de Posición del Borde Rojo (REP) y análisis de series temporales.
* **Estándares:** Alineado con el Estándar Industrial Global para la Gestión de Relaves (GISTM).
